# jdbc
## 需求
### 1、做一个用户系统的增删改查（界面可选），功能包括：添加用户、修改用户、删除单个用户、批量删除用户、根据用户ID查询用户信息、根据条件查询用户翻页列表（查询条件：用户姓名模糊，出生日期，性别）
### 2、用户信息：用户ID、用户名称、性别、身份证号、手机号码、出生日期、家庭住址
## 二、开发工具：idea、maven
## 三、开发语言：java、springboot、restful、mysql，语言版本遵循项目统一要求
